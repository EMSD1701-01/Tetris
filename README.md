### 如何使用？
1. clone项目到您的本地
2. Windows平台：使用任何编译器（如Visual Studio）创建空项目，然后包含src目录和include目录，直接编译即可
3. Linux平台：执行`gcc src/* -I include`，然后执行`./a.out`
